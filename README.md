# Работа с backend'ом

Для начала:
- Скачайте и установите Python 3.11+
- Установите и запустите PostreSQL

## Настройка

Необходимо заполнить все `.gitignor` для бекенда (для каждого есть свой `.exaxmple`).

После проведите миграцию (находясь в корневой папке backend):
```
alembic migrate
```
